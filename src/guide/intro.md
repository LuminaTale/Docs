# 快速开始

本节将帮助你搭建运行环境并且运行你的第一个 Galgame 剧本。

我们将分成以下几个步骤:

1.  **[安装与环境配置](installation.md)**：安装 Rust 编程语言工具链和必要的图形驱动（Vulkan）。
2.  **[快速上手：Hello World](hello-world.md)**：编写你的第一行 Viviscript 脚本，并在屏幕上看到角色向你问好。
3.  **[项目结构说明](project-structure.md)**：了解一个标准的游戏项目包含哪些文件夹，以及把图片、音乐放在哪里。
4.  **[配置文件详解](configuration.md)**：如何修改窗口大小、音量默认值以及开启调试日志。

因为引擎目前仍然处在早期阶段，所以前两个部分会很多和复杂，不过不用担心跟着这份说明一步一步的向下就好了，同时如果真的遇到了未提及的错误，自行去搜索引擎中大部分时间也是可以找到解决方法的。
